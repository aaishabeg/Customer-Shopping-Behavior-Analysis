# Customer-Shopping-Behavior-Analysis

• Analyzed 3,900+ customer transactions to identify purchasing patterns, customer segments, and product performance.
• Applied data preprocessing techniques including missing value treatment, feature engineering, and customer age group
classification.
• Executed complex SQL queries in MySQL to analyze revenue distribution, discount-driven behavior, and subscriber vs.
non-subscriber trends.
• Developed an interactive Power BI dashboard using DAX to visualize category-wise revenue, customer demographics, and
subscription impact.
